# Trimothee CAD Files

This folder contains 3D CAD models and 2D drawings for Trimothee Ultramicrotome Targeting.

Files in this folder are organized as follows:
- CMM_Microtome_BOM.xlsx: Bill of Materials for parts and components.
- CMM_Microtome_SW2023.zip: SolidWorks 2023 assembly and part files.
- *.pdf: 2D drawings and documentation.
