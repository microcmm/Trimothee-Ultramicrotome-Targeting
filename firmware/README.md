# Trimothee Embedded Firmware

This folder contains the embedded firmware code for the Trimothee Ultramicrotome Targeting system. The firmware is designed to run on microcontrollers that interface with the hardware components of the system, such as the AS5048 encoders and communication interfaces.

## Directory Structure
---
- **embedded_control_module/**
  - **src/**: Source code for the firmware, including main application code.
  - **include/**: Header files for the firmware, defining interfaces and constants.
  - **bin/**: Precompiled binary .hex files.

To use this firmware, you will need to:
1. Set up your development environment for the target microcontroller (e.g., Arduino IDE, PlatformIO, etc.).
2. Compile and upload the firmware code to the microcontroller.
